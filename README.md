# 🚀 AI Resume Analyzer

An intelligent web application that analyzes resumes and provides insights like keyword density, skill match, and improvement suggestions.

## 🔥 Features
- 📄 Upload PDF resumes
- 🔍 Extract text automatically
- 🧠 Keyword analysis
- 📊 Skill matching
- 📈 Visual insights (charts)
- ⚡ Fast and simple UI

## 🛠 Tech Stack
- Python (Flask)
- HTML, CSS
- Matplotlib
- PyPDF2
